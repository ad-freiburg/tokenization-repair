Tokenization Repair in the Presence of Spelling Errors

EMNLP 2020 - Reproducibility material for Submission #367

Web demo

A minimalistic web site where you can enter a mistokenized (potentially misspelled) sentence and try multiple of our methods to repair it.

Visit the web demo

Data

The data contains our six benchmarks described in the paper, as well as trained models and predicted sequences from all our methods (1GB compressed). In addition, you can download our training data (6GB compressed).

Download data

Download training data

Code

The code comes with a Docker setup for easy reproducibility. A readme file in the code directory explains how to setup the Docker container. If you are not familiar with Docker, please visit docker.com.

The code allows you to try all our methods in an interactive shell, run them on our benchmarks (or on your own!), and run our evaluation script. Make targets simplify the program calls and give further explanations. It is really fun and simple, we promise!
