Help

Select a correction method and click the button to submit your query. The corrected sequence will be shown below the input form.

Hyperparameters for this demo are set such that the models work best with unreliable spaces (e.g. no spaces at all) - except for "BID the one" and the spelling corrector. Query times are longer than reported in the paper, because the demo is currently set up on a CPU instead of GPU.

Approaches

Name Description
UNI wikipedia Beam search with a forward language model trained on Wikipedia.
BID wikipedia Beam search with a forward language model and a bidirectional model trained on Wikipedia.
BID wikipedia+ Beam search with a forward language model and a bidirectional model trained on Wikipedia with typos.
BID combo+ Beam search with a forward language model and a bidirectional model trained on Wikipedia and arXiv publications with typos and OCR-alike errors.
BID the one BID combo+ with high penalties (will make conservative edits).
spelling correction Beam search with a forward model allowing all kinds of edit operations (beam width = 100).
